# Generalized Angle Division (n-Section): Interactive Model

An interactive geometric construction demonstrating a novel method for dividing any arbitrary angle into *n* equal parts. This model generalizes the trisection method by utilizing the intersection of a circular arc and a specially defined quasi-parabolic curve, governed by the parameter *n*.

## üîç Features
- **Generalized Construction:** Adjust the number of divisions (*n*) and observe the adaptive quasi-parabolic curve.
- **Dynamic Control:** Modify both the original angle (*Œ±*) and the division count (*n*) in real-time.
